# Telugu-Sentiment-Psychosis-Analyzer
This repository implements a language-agnostic analytical pipeline designed to monitor cognitive and emotional shifts through linguistic markers. Specifically optimized for the Psychosis Project, it focuses on analyzing Telugu text to detect structural patterns like Lexical Diversity and Noun-to-Verb Ratios.
The analyzer uses a hierarchical approach to text processing:

Segmentation: Breaking down Telugu agglutinated words into their base components.

POS Tagging: Identifying Nouns and Verbs to calculate the "Structural Health Score."

Alerting: Mathematically flagging subjectivity scores above 0.7 as high internal focus/fixation.
